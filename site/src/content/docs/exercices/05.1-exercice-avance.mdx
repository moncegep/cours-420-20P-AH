## Niveau 5 — Combinaisons avancées

### Exercice 5.1 — Recherche + nettoyage de texte

**Contexte :** Chez *Transport Lacroix*, vous recevez un fichier CSV de commandes de livraison. Les noms des clients sont mal formatés (majuscules, espaces, accents incohérents). Vous devez les croiser avec votre table de clients officielle.

**Feuille `Clients` (données propres) :**

| A — No client | B — Nom | C — Ville | D — Crédit autorisé |
|---|---|---|---|
| CL001 | Tremblay, Michel | Amos | 15 000 $ |
| CL002 | Côté, Nathalie | Val-d'Or | 25 000 $ |
| CL003 | Gagnon, Pierre | Rouyn-Noranda | 10 000 $ |
| CL004 | Bélanger, Julie | La Sarre | 20 000 $ |
| CL005 | Fortin, André | Malartic | 12 000 $ |

**Feuille `Commandes` (données brutes importées) :**

| A — Nom saisi | B — Montant commande |
|---|---|
| `  TREMBLAY, MICHEL  ` | 8 500 $ |
| `côté, nathalie` | 18 200 $ |
| `  GAGNON,   PIERRE` | 12 000 $ |
| `Bélanger, Julie` | 9 800 $ |
| `  FORTIN,ANDRÉ  ` | 7 200 $ |

<Steps>
1. En colonne C, **nettoyez** le nom saisi : `SUPPRESPACE` + `NOMPROPRE` pour standardiser le format.

2. En colonne D, retrouvez le **numéro de client** en utilisant `INDEX`/`EQUIV` avec `MAJUSCULE` et `SUPPRESPACE` appliqués **des deux côtés** de la recherche.

3. En colonne E, retrouvez le **crédit autorisé** du client.

4. En colonne F, affichez « ✓ Approuvé » si le montant de la commande est ≤ au crédit autorisé, sinon « ✗ Dépassement — Approbation requise ».
</Steps>

    ```
    =EQUIV(MAJUSCULE(SUPPRESPACE(A2)); MAJUSCULE(SUPPRESPACE(Clients!B$2:B$6)); 0)
    ```
  </TabItem>

### Exercice 5.2 — Décodage de codes composites

**Contexte :** La brasserie *Boréale* encode chaque lot de production avec un code de 14 caractères : `BOR-IPA-341-06`.

| Position | Contenu | Longueur |
|---|---|---|
| 1–3 | Code brasserie (toujours BOR) | 3 |
| 5–7 | Type de bière | 3 |
| 9–11 | Format (ml) | 3 |
| 13–14 | Quantité par emballage | 2 |

Vous disposez de trois tables de correspondance :

**Table `Types` :** BLA → Blonde d'été, IPA → IPA du Nord-Est, ROU → Rousse, NOI → Noire à l'avoine

**Table `Formats` :** 341 → Bouteille 341 ml, 473 → Canette 473 ml

**Table `Emballages` :** 06 → Pack de 6, 12 → Caisse de 12, 24 → Caisse de 24

**Données :**

| A — Code lot |
|---|
| BOR-IPA-341-06 |
| BOR-BLA-473-12 |
| BOR-ROU-341-24 |
| BOR-NOI-473-06 |

<Steps>
1. En colonne B, extrayez le **code type** (positions 5 à 7) avec `STXT`.
2. En colonne C, retrouvez le **nom de la bière** avec `RECHERCHEX` sur la table Types.
3. En colonne D, extrayez le **code format** et retrouvez la **description du format**.
4. En colonne E, extrayez la **quantité** et retrouvez la **description de l'emballage**.
5. En colonne F, construisez une **description complète** en concaténant : nom + format + emballage.
   Exemple : `IPA du Nord-Est — Bouteille 341 ml — Pack de 6`
</Steps>

<Tabs>
  <TabItem label="💡 Indice">
    ```
    B2: =STXT(A2; 5; 3)
    C2: =RECHERCHEX(B2; Types!A:A; Types!B:B; "Type inconnu")
    D2: =RECHERCHEX(STXT(A2; 9; 3); Formats!A:A; Formats!B:B; "Format inconnu")
    ```
  </TabItem>
  {/* <TabItem label="✅ Solution">
    ```
    B2: =STXT(A2; 5; 3)
    C2: =RECHERCHEX(B2; Types!$A:$A; Types!$B:$B; "Type inconnu")
    D2: =RECHERCHEX(STXT(A2; 9; 3); Formats!$A:$A; Formats!$B:$B; "Format inconnu")
    E2: =RECHERCHEX(DROITE(A2; 2); Emballages!$A:$A; Emballages!$B:$B; "Emballage inconnu")
    F2: =C2 & " — " & D2 & " — " & E2
    ```
    Résultat pour `BOR-IPA-341-06` : **IPA du Nord-Est — Bouteille 341 ml — Pack de 6**
  </TabItem> */}
</Tabs>

### Feuille 6 — `Fiche_Client`

Créez une fiche dynamique (même principe que la fiche rapport de la *Menuiserie Abitibi*) où l'on sélectionne un numéro de client en B1 et toutes les informations se mettent à jour :

```
╔═══════════════════════════════════════════════════════════════╗
║                    LOKAPLEIN AIR — ALMA                       ║
║                   FICHE CLIENT                                ║
╠═══════════════════════════════════════════════════════════════╣
║                                                               ║
║  Client no :         [INDEX/EQUIV]                            ║
║  Nom :               [Prénom Nom — INDEX/EQUIV]               ║
║  Téléphone :         [INDEX/EQUIV]                            ║
║  Courriel :          [INDEX/EQUIV]                            ║
║  Membre :            [INDEX/EQUIV]                            ║
║  Inscrit depuis :    [TEXTE + DATEDIF]                        ║
║                                                               ║
╠══════════════════ HISTORIQUE DE LOCATION ═════════════════════╣
║                                                               ║
║  Nombre de locations :      [NB.SI sur Locations]             ║
║  Montant total loué :       [SOMME.SI sur Locations]          ║
║  Dernière location :        [recherche + MAX]                 ║
║  Équipement le plus loué :  [MODE ou NB.SI par catégorie]     ║
║                                                               ║
╠══════════════════ STATUT ════════════════════════════════════╣
║                                                               ║
║  Locations en cours :       [NB.SI.ENS — pas de retour]       ║
║  Retards en cours :         [calculé]                         ║
║  Total frais de retard :    [SOMME.SI sur Retours]            ║
║  Catégorie client :         [SI basé sur total loué]          ║
║                                                               ║
║  Rapport généré le :  [TEXTE(AUJOURDHUI(); "jj MMMM aaaa")]  ║
╚═══════════════════════════════════════════════════════════════╝
```

<Steps>
1. **Informations du client** : `INDEX`/`EQUIV` sur `Clients`. Formatez la date avec `TEXTE`. L'ancienneté d'inscription utilise `DATEDIF`.

2. **Historique** : `NB.SI` et `SOMME.SI` sur `Locations` filtrés par le numéro de client en B1. Pour la dernière location, combinez `MAX` + `SI` en matriciel ou utilisez `RECHERCHEX` avec le mode de recherche « dernier ».

3. **Statut** : Croisez les feuilles `Locations` et `Retours` pour identifier les locations non retournées. Catégorisez le client selon son total de locations :
   - < 200 $ → « Occasionnel »
   - < 500 $ → « Régulier »
   - ≥ 500 $ → « Fidèle »

4. **Mise en forme** : Format professionnel avec bordures, couleurs et prêt à imprimer sur une page.
</Steps> */}
