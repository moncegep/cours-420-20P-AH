## Pourquoi utiliser la fonction SI ?

La fonction SI permet de prendre des décisions dans vos feuilles de calcul en fonction de conditions spécifiques. 
Elle est essentielle pour créer des formules dynamiques qui s'adaptent aux données saisies.

## Rappel des opérateurs de comparaison

Voici les principaux opérateurs de comparaison disponible dans Excel:


| Opérateur   | Description               | Exemple        |
|-------------|---------------------------|----------------|
| `=`         | Égal à                    | `A1=10`        |
| `<>`        | Différent de              | `A1<>10`       |
| `>`         | Supérieur à               | `A1>10`        |
| `<`         | Inférieur à               | `A1<10`        |
| `>=`        | Supérieur ou égal à       | `A1>=10`       |
| `<=`        | Inférieur ou égal à       | `A1<=10`       |


Ils sont utilisés dans la fonction `SI` pour exprimer le `test_logique` sous forme de comparaison entre des valeurs ou des cellules.

## Syntaxe de la fonction SI

La fonction SI a la syntaxe suivante :

```excel
=SI(test_logique; valeur_si_vrai; valeur_si_faux)
```
Elle nécessite trois arguments :
- `test_logique` : une condition qui peut être VRAI ou FAUX. 
    - Généralement le test prend la forme d'une comparaison entre des valeurs ou des cellules.
    - Exemple: `A1>10` teste si la valeur dans la cellule A1 est supérieure à 10.
- `valeur_si_vrai` : la **valeur** ou **formule** à retourner si le test est VRAI
- `valeur_si_faux` : la **valeur** ou **formule** à retourner si le test est FAUX


<Aside type="caution" title="Attention: Séparateurs dans les formules">
Dans la **version française** d'Excel, les **arguments d'une formule sont séparés** par des **points-virgules (;)** au lieu de virgules (,).  
Vérifiez toujours le séparateur correct selon la langue de votre version d'Excel. 
Il est visible dans la barre de formule lorsque vous tappez la formule.
</Aside>

## Exemples simples: Affichage conditionnel

### Exemple 1 : Vérifier l'âge

```excel
=SI(A1>=18;"Majeur";"Mineur")
```
Si la valeur dans A1 est 20, le résultat sera "Majeur". Si A1 est 16, le résultat sera "Mineur".

### Exemple 2 : Appliquer une remise

```excel
=SI(A1>100;A1*0,9;A1)
```
Si le montant dans A1 est 150, le résultat sera 135 (10% de remise). Si A1 est 80, le résultat sera 80 (pas de remise).

## SI imbriqué

Comme indiqué plus haut, lorsqu'on utilise la fonction SI, on indique la `valeur_si_vrai` qui peut etre simplement une valeur à afficher ou 
de préférence une formule à appliquer. C'est dans ce cas que l'on peut imbriquer un second SI pour faire un test supplémentaire.

La fonction SI peut être imbriquée pour gérer plusieurs conditions.

### Exemple 3 : Classification des notes

Si on veut classer les notes en "Fort", "Moyen", "Faible" selon les critères suivants :
- "Fort" si la note ≥ 80
- "Moyen" si la note ≥ 60
- "Faible" sinon
On peut utiliser la formule suivante :

```text
=SI(A1>=80;"Fort";SI(A1>=60;"Moyen";"Faible"))
```

Si on suppose que la formule est dans la cellule `B1` et que `A1` contient la note, alors :
- Si A1=85, le résultat affiché dans B1 sera "Fort"
- Si A1=70, le résultat affiché dans B1 sera "Moyen
- Si A1=50, le résultat affiché dans B1 sera "Faible"

## Autres fonctions conditionnelles

En plus de la fonction SI, Excel propose plusieurs autres fonctions pour gérer des conditions :

- `NB.SI` : Compte le nombre de cellules qui répondent à un critère spécifique.
- `SOMME.SI` : Additionne les valeurs qui répondent à un crit
- `MOYENNE.SI` : Calcule la moyenne des valeurs qui répondent à un critère spécifique.
- `ET` : Vérifie si toutes les conditions sont vraies.
- `OU` : Vérifie si au moins une condition est vraie.

Ces fonctions sont très utiles pour analyser des données en fonction de critères spécifiques.
